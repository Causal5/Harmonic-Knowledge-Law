# Examples

This directory contains worked examples showing the Causal Ethics / HKL / SLP framework in application.

Worked examples should demonstrate how the framework behaves when applied to a concrete system, instead of only defining theoretical primitives.

## Canonical Example Slots

- `sahel-worked-example.md` — Sahel application case study. Intended to demonstrate framework use against a real regional ecological / infrastructural / civilizational constraint problem.

## Purpose

Examples should make the framework testable by showing:

1. System boundary selection.
2. Relevant causal constraints.
3. Burden functions or observable destabilization terms.
4. Corrective-effort logic.
5. HKL stability interpretation.
6. OIE / PAE or related effort accounting.
7. Limits, falsifiability, and failure modes.
