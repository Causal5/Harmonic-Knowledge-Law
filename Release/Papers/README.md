## Publication Standard

A paper in this directory should ideally include:

1. Abstract or framing summary.
2. Symbol dependencies.
3. Core claim.
4. Definitions.
5. Argument or derivation.
6. Objections / limitations.
7. Relation to HKL / CE / SLP system architecture.
