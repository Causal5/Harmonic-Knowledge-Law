# Release/archive — legacy duplicates

Files in this folder are older, byte-for-byte duplicates of documents that now live in the organized Release subdirectories: Papers/, Delta-Self/, HKL/, Registry/, and Examples/.

They are preserved here for historical continuity and are not independently maintained. Where a document appears both here and in an organized subdirectory, the subdirectory copy is canonical.

Two archived files are older editions whose newer versions are canonical elsewhere: the Delta-Self worldline formalization and the master symbol registry.
